## Станция "Функционально-Лингвистическая"

Итак функция объявляется словом def - от англ. definition(определение). На русском это "опр. - как и def три буквы".

На самом деле это озночает что функция лишь определенина(но я бы назвал бы объявлена), что вроде как и в математике озночает что она есть но не обязательно изпользована, изпользование функции называется вызовом

Следует помнить что по какой то неводомой и странной, противоречащий здравому смыслу задумке, в python пишется не

```js
function name() {
    const a = 2;
}
```

```python
def name():
    a = 2
```

Согласен с тем что ":" это крайне неудобно

Вызываются эти функции:

```python
def name
```

И вызывают они функцию очень странно, с пробелом:

name ()

## "Чистая функция"

1. Не вводит и не выводит ничего(input и print)
2. Не записывает ничего в глобальные переменные, не заглядывает в эти глобальные переменные

## Рекурсия

Пример: сумма цифр числа 1234. Если человек разумный, не самоубийца то вы скорее всего будете изпользовать цикл, или там что то подобное.

но если вы совсем отбитые, то вы будете изпользовать рекурсию. пример:

```py
number = 1234
sum = 0
def a(parametr):
    sum = sum + parametr % 10
    return parametr // 10
a(number)
```

Здесь функция `a` которая вместо того чтобы считать сумму цифр, она отделяет последнее число и кидает его в сумму
